sisters_and_parent

Description

Return a list with tuples that join both sister leaves with its parent (including, the word right/left/parent), the node id and the name (only for leaves). See the test.

Tests

    >>> sisters_and_parent(newick_file2nodes("tree_abcdef.dnd"))
    [(('left', 5, 'B'), ('right', 4, 'A'), ('parent', 3)), (('left', 8, 'D'), ('right', 7, 'C'), ('parent', 6)), (('left', 11, 'F'), ('right', 10, 'E'), ('parent', 9))]
