node2leaves

Description

Return a list with the names of all leaves that can be reached from a node i_node. The function should call itself in a recursive way. The idea is to start exploring left then rigth. Takes as input the node id to start the recursion, a dictionary of nodes and the list of leaves which initially should be emtpy.
