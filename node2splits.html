node2splits

Description

This function should call itself in a recursive way. The idea is to start exploring left then rigth. Takes as input the node id to start the recursion, a dictionary of nodes. The function should 1) print the left and right lists and 2) return the concatenation of both lists.

Tests

    >>> node2splits(1, newick_file2nodes("tree_abcdef.dnd"))
    ['F'] ['E']
    ['D'] ['C']
    ['B'] ['A']
    ['D', 'C'] ['B', 'A']
    ['F', 'E'] ['D', 'C', 'B', 'A']
    ['F', 'E', 'D', 'C', 'B', 'A']
